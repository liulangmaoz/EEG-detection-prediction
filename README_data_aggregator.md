# 脑电数据汇总脚本使用说明

## 功能介绍

`data_aggregator.py` 是一个用于汇总脑电数据处理结果的脚本，它能够：

1. **自动检测并处理** 癫痫数据和正常数据的结果文件
2. **收集** 脑电主频功率、功率谱熵、各频段能量功率密度与相对占比，以及四个混沌指数（D2、PD2、LE、KE）的值
3. **汇总** 所有数据到一个Excel文件中，按照类别（癫痫/正常）和阶段（发作前期/发作期/发作后期/正常时期）分好类
4. **支持** 自动创建输出目录，确保结果文件能够正确保存

## 目录结构

脚本默认的目录结构如下：

```
├── LFP.py                    # 癫痫数据处理脚本
├── LFP_non.py                # 正常数据处理脚本
├── data_aggregator.py        # 数据汇总脚本
├── data/
│   ├── ictal_epilepsy/       # 癫痫数据输入目录
│   └── interictal_normal/    # 正常数据输入目录
└── results/
    ├── ictal_epilepsy/       # 癫痫数据结果目录
    ├── interictal_normal/    # 正常数据结果目录
    └── aggregated/           # 汇总结果保存目录
        └── eeg_data_aggregated.xlsx  # 汇总结果文件
```

## 使用步骤

1. **运行数据处理脚本**：
   - 首先运行 `LFP.py` 处理癫痫数据
   - 然后运行 `LFP_non.py` 处理正常数据
   - 这两个脚本会在 `results` 目录下生成相应的结果文件

2. **运行数据汇总脚本**：
   ```bash
   python data_aggregator.py
   ```

3. **查看结果**：
   - 汇总结果会保存到 `results/aggregated/eeg_data_aggregated.xlsx` 文件中
   - Excel文件包含多个工作表：
     - `所有数据`：包含所有汇总数据
     - `癫痫数据`：仅包含癫痫数据
     - `正常数据`：仅包含正常数据
     - 各个阶段的单独工作表：如 `发作前期`、`发作期`、`发作后期`、`正常时期`

## 数据字段说明

汇总数据包含以下字段：

### 基本信息
- `文件编号`：数据文件的编号
- `阶段`：数据所属的阶段（发作前期/发作期/发作后期/正常时期）
- `类别`：数据类别（癫痫/正常）

### 能量占比（各频段）
- `DELTA能量_avg`：DELTA频段能量平均值
- `DELTA占比_avg`：DELTA频段能量占比平均值
- `THETA能量_avg`：THETA频段能量平均值
- `THETA占比_avg`：THETA频段能量占比平均值
- `ALPHA能量_avg`：ALPHA频段能量平均值
- `ALPHA占比_avg`：ALPHA频段能量占比平均值
- `BETA能量_avg`：BETA频段能量平均值
- `BETA占比_avg`：BETA频段能量占比平均值
- `GAMMA能量_avg`：GAMMA频段能量平均值
- `GAMMA占比_avg`：GAMMA频段能量占比平均值

### 主频功率
- `α主频_avg`：α波主频平均值
- `α功率_avg`：α波功率平均值
- `α总功率_avg`：α波总功率平均值
- `β主频_avg`：β波主频平均值
- `β功率_avg`：β波功率平均值
- `β总功率_avg`：β波总功率平均值
- `α+β主频_avg`：α+β波主频平均值
- `α+β功率_avg`：α+β波功率平均值
- `α+β总功率_avg`：α+β波总功率平均值

### 功率谱熵
- `总功率谱熵_avg`：总功率谱熵平均值
- `归一化总熵_avg`：归一化总熵平均值
- `α波熵_avg`：α波熵平均值
- `归一化α熵_avg`：归一化α熵平均值
- `β波熵_avg`：β波熵平均值
- `归一化β熵_avg`：归一化β熵平均值

### 混沌指数
- `D2_avg`：D2关联维数平均值
- `PD2_avg`：PD2点关联维平均值
- `LE_avg`：LE李雅普诺夫指数平均值
- `KE_avg`：KE科尔莫戈罗夫熵平均值

## 注意事项

1. **数据格式**：脚本会自动检测结果文件的格式，确保与最新的LFP.py和LFP_non.py输出格式兼容

2. **错误处理**：脚本包含错误处理机制，即使某些文件不存在或格式不正确，也能继续处理其他文件

3. **路径配置**：脚本使用相对路径，确保在不同环境下都能正常运行

4. **数据缺失**：对于缺失的数据，脚本会使用NaN值填充，确保数据结构的完整性

## 依赖项

- Python 3.6+
- pandas
- numpy
- openpyxl（用于读取和写入Excel文件）

## 运行示例

```bash
# 处理癫痫数据
python LFP.py

# 处理正常数据
python LFP_non.py

# 汇总数据
python data_aggregator.py
```

运行完成后，您可以在 `results/aggregated/eeg_data_aggregated.xlsx` 文件中查看汇总结果。
